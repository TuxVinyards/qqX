qqX - quickemu quickget X terminal project  


A independent text based menu interface designed to work with QEMU & with the 'quickemu' project

https://github.com/quickemu-project/quickemu   https://gitlab.com/qemu-project/qemu

Users should install 'qemu' & 'quickemu' separately, as normal.

#

ffX - the ff(mpeg) X-terminal project - AV processing scripts for ffmpeg
