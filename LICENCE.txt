qqX - quickemu quickget X terminal project   

2023/07   Scripts based on ffX & designed to work with 'quickemu-wrap'

ffX - the ff(mpeg) X-terminal project - AV processing scripts for ffmpeg   See:  https://github.com/TuxVinyards

Any references to the ffX project imply GPL3 and its transfer between the projects


A text menu interface for the 'quickemu' project:

https://github.com/quickemu-project/quickemu   https://gitlab.com/qemu-project/qemu

Users should install 'qemu' & 'quickemu' and may set up Virtual Machines as normal.
