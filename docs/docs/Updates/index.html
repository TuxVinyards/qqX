Updates

qqX will alert you if a new release is available​

A highlighted [dl] option will appear on the qqX menu pages when a new release is available.

  • The installer script will check that new features get added to your settings file

  • See the settings file for control of alerts for beta releases and for custom download folders

You can also download directly from the qqX repository and run the installer from the command line or a file manager.

  • If newer branches are present, they may have the latest tweaks or features, and may possibly have bug fixes. On the web pages, switch branches, click on the 'code' button and 'download zip'.

All locations and settings preferences will be kept​
