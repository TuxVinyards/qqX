Downstream fixes for Quickemu​

Creative differences​

There are always creative differences within projects.

The Quickemu project is no exception and the growing count of developers that walked away during 2024 should not affect you as a user.

Relationship with Quickemu​

More independence means that you get a faster moving program that is not being held back.

Meanwhile, qqX will continue to support virtual machines that have been created using QuickEmu or QuickGUI.

There used to be a qqx listing on the Quickemu wiki pages but things need to move on and need to evolve.
